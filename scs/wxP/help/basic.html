Not working?

  1. First make sure all your vales correct

Specific game not working?

  1. Other applications, such as some games, may register hooks that swallow all key events.

Requirements

  • Process name: This does not need the full name - EX: notepad equals any process that has notepad in the name
  • Cheats file: The location of your cheats file - This is the same for all systems
  • Game name: The name of the game to load the cheats for - This is defined in the cheats file
  • Teminate shortcut: The shortcut to deactivate all shortcuts - Important: This does not close the app only removes shortcuts internally

On Windows

  • Process name: Can be found on task manager

  • Limitations

    • Events generated under Windows don't report device id *(all keyboards connected are the same)

On Linux

  • This app requires root permissions on linux explained below
  • Process name: Can be found on task manager

  • Limitations

    • Media keys on Linux may appear nameless
    • To avoid depending on X, "keyboard" *(Python module) reads raw device files
